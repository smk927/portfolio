DevCanvas - Automated Software Documentation Tool

Visit GitHub Link

DevCanvas is an automated tool designed to simplify software documentation for Python and Java. It allows users to generate key documentation elements such as class diagrams, sequence diagrams, flowcharts, and summaries in just three steps: upload the code, select documentation type, and generate. The tool utilizes Abstract Syntax Trees (AST) and integrates with PlantUML and Google's GenAI API to produce accurate, structured, and efficient documentation automatically.

The full-stack application, built with React on the frontend and Django on the backend, offers a seamless user experience. It also includes secure email-based authentication, enabling users to manage and download generated documentation with ease. DevCanvas streamlines the documentation process, saving time and improving workflow for developers.
